# GALAXIAN / Space Invaders

Galaxian on nodejs.

Install globally or run using npx

`npm i -g galaxian`

or

`npx galaxian`

and play the game.

## Controls

| Key      | Action                |
| -------- | --------------------- |
| LEFT     | Moves Spaceship Left  |
| RIGHT    | Moves Spaceship Right |
| SPACE    | Fires bullets         |
| S        | Shows Monster stats   |
| L        | Shows Mini log        |
| CTRL + C | Exits Game            |
